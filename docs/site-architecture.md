# サイト構造

## グローバルナビ

```
ホーム ／ 記事一覧 ／ 特集 ／ Q&A ／ CAREER EDITとは
```

加えて、検索アイコンをヘッダー右上に常駐。

---

## ページ構成

### メインページ

| ページ | URL（想定） | 役割 |
|---|---|---|
| ホーム | `/` | メディアの入口。最新記事、ランキング、主要導線 |
| 記事一覧 | `/articles/` | 全記事のカテゴリ別・タグ別フィルター |
| 特集 | `/features/` | 連載・企画パッケージの一覧 |
| Q&A | `/qa/` | 読者からの質問と編集部の回答 |
| CAREER EDITとは | `/about/` | メディアの理念・編集方針・編集部紹介 |

### 記事詳細ページ

| ページタイプ | URL（想定） | 特徴 |
|---|---|---|
| 企業情報 | `/company/{slug}/` | タブ構造、データ表、編集部見解、関連記事 |
| 職種図鑑 | `/job/{slug}/` | 職種解説、キャリアパス、年収レンジ |
| キャリアインタビュー | `/interview/{id}/` | 実名・顔出し取材。リード文付き |
| 現場の声 | `/voice/{id}/` | 匿名・短尺。編集部コメント付き |
| 編集部記事 | `/note/{id}/` | 編集部・編集長の視点記事 |
| 転職者の声 | `/transition/{id}/` | 転職経験者のインタビュー |

### ユーティリティページ

| ページ | URL（想定） | 役割 |
|---|---|---|
| 検索結果 | `/search?q=` | キーワード検索＋カテゴリ絞り込み |
| タグアーカイブ | `/tag/{slug}/` | 特定タグの記事一覧 |
| 相談フォーム | `/contact/` | 読者からの相談受付 |
| 運営会社 | `/company-info/` | マルサン情報 |
| 利用規約 | `/terms/` | |
| プライバシーポリシー | `/privacy/` | |
| お問い合わせ | `/inquiry/` | 取材・問い合わせ全般 |

## ホームページの構成

```
┌─ ヘッダー（ロゴ + グローバルナビ + 検索アイコン）
│
├─ ヒーローエリア
│  ・事業説明コピー
│  ・CTA「→ キャリアの悩みを、編集部に相談する」
│
├─ CONTENTS（メインエリア + サイドバー）
│  ・メイン：最新記事リスト（カテゴリラベル・日付・タグ付き）
│  ・サイド：RANKING / CATEGORY / TAG
│
├─ ページネーション
│
└─ フッター
   ・お問い合わせ / 利用規約 / プライバシーポリシー / 運営会社
   ・Copyright © MARUSAN Co., Ltd.
```

## 記事詳細ページの共通構造

どの記事タイプでも共通する要素：

- **ヘッダー**：グローバルナビと同じ
- **パンくず**：記事一覧 ＞ カテゴリ ＞ 記事タイトル
- **メタ情報**：カテゴリ・日付・記事 ID
- **タイトル**
- **リード文**（雑誌風のフックとなる導入）
- **著者・取材対象者情報**
- **本文**（見出し付き）
- **タグ**（記事末尾に 3〜5 個）
- **関連記事**（カテゴリ・タグベースで自動生成）
- **相談 CTA**（Q&A への誘導）

## 企業情報ページの詳細構造

企業情報は CAREER EDIT の主要コンテンツ。70 項目相当のフィールドを管理するが、読者には段階的に見せる設計。

### タブ構造

```
[会社を知る] [インタビュー] [現場の声] [年収・待遇] [選考情報]
```

### 会社を知るタブの中身

1. **会社概要**（設立、代表、所在地、従業員数などの基本情報）
2. **編集部の見解**（業界内での立ち位置、活躍する人の特徴）
3. **年収レンジ**（年代別・職種別の相場）
4. **キャリアインタビュー**（その企業の実名取材記事へのリンク）
5. **現場の声**（その企業の匿名取材記事へのリンク）

詳細なフィールド設計は別途 [`company-page-fields.md`] で管理予定。

## Q&A コーナーの構造

```
┌─ ヒーロー（「編集部に、相談する。」＋ 2 つの CTA）
│
├─ 連載「編集長に聞く」（最新エピソード）
│
├─ カテゴリで探す（6 カテゴリのグリッド）
│  ・キャリアの選択肢
│  ・業界選び
│  ・年収・処遇
│  ・転職のタイミング
│  ・面接・選考
│  ・副業・独立
│
├─ 最新の問答（4〜5 本）
│
└─ 質問を投稿する（CTA）
```

詳細は [`docs/qa-design.md`] で別途。

## モバイル対応

- レスポンシブ対応必須
- スマホでは左ナビは折りたたみ（ハンバーガーメニュー）
- 企業情報ページのタブは横スクロール
- 記事詳細は縦リーダビリティ優先
