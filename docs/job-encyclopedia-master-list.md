# 職種図鑑マスターリスト

CAREER EDIT の職種図鑑で扱う職種の全体像と優先順位を整理したドキュメント。

**最終更新**：2026年4月24日

---

## 方針

- **目標**：約70職種をカバー（マルニ職種辞典の34職種をベースに拡張）
- **立ち上げ戦略**：まず10-15職種を看板記事として深く書き、残りは順次追加
- **独自性**：マルさんの経験・視点を活かした独自職種を織り込む

## 優先度の層

### 第1層：看板記事（最優先・10-15職種）

立ち上げ時の顔になる記事。CAREER EDIT のクオリティを示す。

**実装済み（5本）**
- コピーライター
- マーケター（事業会社）親
- マーケター（BtoB / SaaS）子
- クリエイティブディレクター
- データサイエンティスト

**次に書くべき（5-10本）**
- ストラテジックプランナー（マルニ人気1位）
- プロダクトマネージャー（PdM）
- Webディレクター
- マーケター（BtoC）
- マーケター（スタートアップ）
- アートディレクター
- 編集型マーケター（マルさん独自）
- 地域ブランドプロデューサー（マルさん独自）

### 第2層：中堅記事（30-40職種）

網羅的にカバー。マルニと同レベル。

### 第3層：ニッチ記事（20-25職種）

検索ニーズは少ないが、網羅性と専門性のため存在。

## 量産の進め方

### フェーズ1（立ち上げ時）：看板記事 10-15本を集中執筆

- 1記事あたり30分-50分（Claude活用）
- マルさん1人で1週間集中執筆で達成可能

### フェーズ2（半年以内）：第2層を順次追加

- 取材データを織り込みながら品質を上げる
- 月5-10本ペース

### フェーズ3（1年以内）：第3層の網羅完了

- 残りのニッチ職種を追加
- 既存記事の深掘り版への更新も並行

## 運用ルール

### 記事の構造

全職種で**17セクション構成**を統一。親子構造の場合は親記事と子記事で重複を避ける。

### 7業態分類

広告・クリエイティブ系職種（コピーライター、CD、プランナー、デザイナー、ディレクター等）では、**7業態分類**を統一して使う。

### 年収データ

2026年時点の編集部算定値として明記。年次で更新する。

### 編集部見解

各職種の「2026年現在の実態」「業態別の違い」「編集部から」セクションで、マルさん・編集部の見解を明示する。
