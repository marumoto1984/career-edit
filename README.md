# CAREER EDIT

広告・マーケ・IT領域で、転職・キャリアを検討されている方へ豊かな情報を提供するメディア。

> **キャリアは、編集できる。**

---

## このリポジトリについて

CAREER EDIT の立ち上げにあたり、ブランド・サイト設計・コンテンツ方針・職種図鑑を整理・議論するための作業用リポジトリです。

- **運営会社**：株式会社マルサン（福岡県福岡市中央区天神）
- **編集長**：丸本 翔一
- **ドメイン**：careeredit.jp
- **ステータス**：設計・プロトタイプフェーズ（2026年4月〜）

### 信頼できる方限定でご覧ください

このリポジトリは技術的には Public ですが、**信頼関係のある方にのみURLを共有する前提**で運用しています。未発表のブランド・事業戦略・モック資料を含むため、外部への転載・拡散はご遠慮ください。

## このメディアの立ち位置

**既存のキャリアメディアとの決定的な違い**：

- 人材紹介業のオウンドメディアではなく、**独立編集メディア**
- 特定企業・エージェントと利害関係がないため、**業界の実態に踏み込んで書ける**
- 編集部の見解を前面に出した、**主張のあるキャリアメディア**
- AI時代の業界変化を正直に扱う

### 4つの競合（マルニ、マスメディアン advanced、CREATIVE VILLAGE、MarkeZine）との差別化

| 要素 | CAREER EDIT | 他社 |
|---|---|---|
| 編集部の見解の強さ | ⭐⭐⭐⭐⭐ | 中立気味 |
| 業界構造への踏み込み | ⭐⭐⭐⭐⭐ | 利害相反で書けない |
| 業態の精密な分類 | ⭐⭐⭐⭐⭐ | 粗い括り |
| AI時代への対応 | ⭐⭐⭐⭐ | 部分的 |
| 6カテゴリ相互連携 | ⭐⭐⭐⭐ | なし |

## 職種図鑑の構造

### 17セクション構成（全職種共通）

```
I. OVERVIEW / この職種を知る
  01. 概要
  02. 2026年現在の実態

II. WORK / 業務の実態
  03. 業務内容
  04. 業態別の違い（7業態比較）
  05. 1つの仕事の流れ
  06. 代表的な成果事例

III. DATA / データで見る
  07. 年収レンジ
  08. キャリアパス

IV. FUTURE / AI時代を生きる
  09. AI時代のスキルセット

V. ENTRY / この職種になる
  10. なるためには
  11. 選考・採用プロセス
  12. 継続的なスキルアップ

VI. Q&A / よくある質問
  13. FAQ（15問）

VII. DEEP DIVE / 深める
  14. おすすめの書籍・リソース
  15. 関連記事
  16. 活躍する企業

VIII. FROM EDITOR / 編集部から
  17. この職種を検討する方へ
```

### 7業態構成（広告・クリエイティブ系職種）

| 業態 | 主な該当企業 |
|---|---|
| 大手総合代理店 | 電通、博報堂 |
| 総合代理店 | ADK、東急エージェンシー、読売広告社、大広、I&S BBDO |
| 独立系ブティック | GO、TUGBOAT、チョコレイト、箱、enjin、Party |
| デジタル代理店 | 電通デジタル、サイバーエージェント、セプテーニ、アイレップ |
| 制作会社 | 博報堂アイ・スタジオ、TYO、AOI Pro.、エージー |
| 事業会社（インハウス） | 各業界 |
| フリーランス | - |

### 親子構造（マーケター系）

業界セグメントで仕事が大きく異なる職種は親子構造を採用：

```
マーケター（事業会社・親）
├ マーケター（BtoB / SaaS）← 実装済み
├ マーケター（BtoC）       ← Coming Soon
├ マーケター（D2C）        ← Coming Soon
├ マーケター（スタートアップ）← Coming Soon
└ マーケター（メーカー）    ← Coming Soon
```
