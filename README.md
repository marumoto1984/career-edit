# CAREER EDIT

広告・マーケ・IT領域で、転職・キャリアを検討されている方へ豊かな情報を提供するメディア。

> **キャリアは、編集できる。**

---

## このリポジトリについて

CAREER EDIT の立ち上げにあたり、ブランド・サイト設計・コンテンツ方針を整理・議論するための作業用リポジトリです。

- **運営会社**：株式会社マルサン（福岡県福岡市中央区天神）
- **ドメイン**：careeredit.jp
- **ステータス**：設計フェーズ（2026年4月〜）

### 信頼できる方限定でご覧ください

このリポジトリは技術的には Public ですが、**信頼関係のある方にのみURLを共有する前提**で運用しています。未発表のブランド・事業戦略・モック資料を含むため、外部への転載・拡散はご遠慮ください。

## モック

現在公開している主要な成果物です。

### サイト全体モック

📄 [`index.html`](./index.html) / [公開URL](https://marumoto1984.github.io/career-edit/)

10ページ分のサイトモックを1つのファイルにまとめたもの。左側の黒いナビから各ページを切り替えできます。

収録ページ：
- ホーム
- 記事一覧
- 特集
- Q&A
- CAREER EDITとは（About）
- 企業情報詳細
- 職種図鑑詳細
- キャリアインタビュー詳細
- 現場の声詳細
- 検索結果

> ⚠️ これは**レビュー用モック**です。本番サイトは各ページが独立したURLで動作します。モック内の記事・人物・会話はすべて架空のダミーコンテンツです。
